# ReUnfold-Square — презентація

HTML-презентація проєкту розрахунку площі розгортки листового металу.

## Як відкрити
Локально: відкрити `index.html` подвійним кліком (папка `renders/` має лежати поруч).

## GitHub Pages (посилання для показу)
1. Settings → Pages
2. Source: Deploy from a branch → Branch: `main` → Folder: `/ (root)` → Save
3. За 1-2 хв зʼявиться посилання виду `https://<user>.github.io/<repo>/`

Керування: стрілки / пробіл — гортання, «Авто» — автопрокрутка, UA/RU — мова.
